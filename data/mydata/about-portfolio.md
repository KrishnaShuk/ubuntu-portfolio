# About This Portfolio

Welcome to my interactive portfolio! This project is a **pixel-perfect simulation** of the Ubuntu Desktop Environment, built to run seamlessly in your browser. It was created not just to showcase my projects, but to demonstrate my passion for building complex, high-fidelity user interfaces.

---

### Core Technologies

This entire OS simulation was built from the ground up using a modern, performant, and type-safe tech stack:

*   **Framework:** Next.js (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **State Management:** Zustand
*   **Animation:** Framer Motion

---

### How to Use It

*   **Draggable Windows:** Click and drag any window by its top header bar.
*   **Resizable Windows:** Drag the corners or edges of any window to resize it.
*   **Maximize:** Double-click a window's header or drag it to the top edge of the screen.
*   **Explore the Dock:** Click the icons on the left to launch different simulated applications, like the Terminal or VS Code.
